# Pixora

Pixora is an iOS app that aggregates free image APIs and provides a unified interface
to search and download images from multiple sources.

## Features (MVP)
- Multiple image providers
- Unified search
- Image download with attribution
- No accounts, no tracking

## Tech
- UIKit
- iOS 15+
- Swift
- GCD / OperationQueue
- No third-party dependencies
